# Sample Docs

This folder contains multiple markdown examples you can render with the app.

- [Quick Reference](./quick-reference.md): small mixed-content sample (headings, table, Mermaid, and code).
- [Codeblocks Playground](./codeblocks-playground.md): language-focused code block examples.
- [System Design Walkthrough](./system-design-walkthrough.md): larger end-to-end sample with diagrams, snippets, and structure.

Use this folder as your docs root:

```bash
go run . -dir ./samples
```
